## ❓ Troubleshooting

### App will not open

- Make sure you downloaded the Windows file.
- Try right-clicking the file and choosing Run as administrator.
- Check if Windows blocked the file.
- Download it again if the file looks incomplete.

### Windows says the app is unsafe

- This can happen with files from GitHub.
- Open the file again and choose More info if Windows shows that option.
- Then choose Run anyway if you trust the source.

### Nothing happens after I start it

- Wait a few seconds.
- Check if the app opened behind other windows.
- Open Task Manager and see if the app is running.
- Try restarting the app.

### The request does not work

- Check your API key.
- Check your network connection.
- Make sure the model name is correct.
- Try again with a simpler prompt.
